Welcome to Destination Sol

Destination Sol is a space arcade shooter that we (the MovingBlocks open source organization) adopted in early 2015.

Running Destination Sol from source

Requires Java 7 or 8. Can either use Git or download a source zip from GitHub (link above)

$ git clone https://github.com/MovingBlocks/DestinationSol.git
$ cd DestinationSol
$ gradlew run
